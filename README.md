MiyaGantt-v1.0.0

「がんすけ」等と同等の直感的な操作が可能な、高機能ガントチャート作成ツールです。
マウスで直接チャートを操作し、プロジェクトのスケジュールを素早く計画・管理できます。

主な機能

1. マウス操作によるタスク作成
   チャートの空きスペースを右にドラッグするだけで、その期間に新しいタスクを追加できます。

2. 直感的なバー操作
   - 移動: バーを掴んで左右にドラッグすると、日付を変更できます。
   - 期間の変更: バーの右端または左端をドラッグすると、期間を自在に伸縮できます。
   - 詳細の編集: テーブルの行、またはチャートのバーを「ダブルクリック」すると、タスクの編集画面が開きます。

3. 進捗率（プログレスバー）とカラー変更
   タスク編集画面から「進捗率(%)」と「バーの色」を設定可能です。
   バーの内部に進捗率が視覚的（プログレスバー）に表示され、ひと目でタスクの状況と重要度が把握できます。

4. ズーム機能と「今日」の表示
   - 上部のスライダーで、チャートの表示倍率（1日の横幅）を自由に変更できます。
   - チャート上には「Today」を示す赤い点線が引かれ、現在の進捗と計画のズレを瞬時に確認できます。

5. 週末の視覚化
   土曜日（薄い青）と日曜日（薄い赤）が自動的に強調表示され、休日の計画が立てやすくなっています。

動作環境

Windows 11 / Windows 10

インストール・実行方法

配布された MiyaGantt-v1.0.0.exe を任意のフォルダに配置します。

MiyaGantt-v1.0.0.exe をダブルクリックして起動してください。

※初回起動時の注意（Windows Defenderの警告が出た場合）
Windowsのセキュリティ機能（SmartScreen）により、青い警告画面「Windows によって PC が保護されました」が表示される場合があります。
これは新規に作成されたアプリでよく発生する画面です。以下の手順で起動してください。

1. 警告画面のテキスト内にある 「詳細情報」 をクリックします。
2. 右下に現れる 「実行」 ボタンをクリックすると、アプリが起動します。

基本的な使い方

- 新規作成: チャート上の何もない場所をドラッグ操作するか、左上の「追加」ボタンを押します。
- 編集: タスクをダブルクリックして、名前、期間、進捗率、色を変更します。
- 削除: チャート上のバーを右クリックコンテキストメニューから削除するか、左側の表で選択して「削除」ボタンを押します。
- 保存: 「保存」ボタンで現在の状態をファイル（.json形式）に保存します。
- 読込: 「読込」ボタンで以前保存したプロジェクトを開きます。
