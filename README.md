📊 MiyaGantt v2.0.0

「がんすけ」等と同等の直感的な操作が可能な、高機能ガントチャート作成ツールです。
マウスで直接チャートを操作し、プロジェクトのスケジュールを素早く計画・管理できます。

✨ 主な機能

🖱️ 移動: バーを掴んでドラッグすると、日付の変更や、上下の別のタスク（行）への移動が自由にできます。複数のバーをまとめて移動させることも可能です。

🔳 範囲選択: チャートの何もない場所をドラッグして、範囲内のバーを一括選択できます（Ctrlキー＋クリックで個別選択も可能）。

➕ 直感的なタスク作成: チャートの空きスペースを Shiftキー を押しながらドラッグするだけで、その期間に新しいタスクを追加できます。

↔️ 期間の変更: バーの右端または左端をドラッグして、期間を自在に伸縮できます。

📝 バーのテキスト編集: バーをダブルクリックすると、バーごとに個別のテキスト（名前）を入力・表示できます。

✍️ インライン編集: 左側のテーブルの各セルをダブルクリックして、直接テキストを入力しタスクを編集できます。

⚡ 期間の追加（ショートカット）: タスク行の何もない場所をダブルクリックすると、その位置に1日分の期間を素早く追加できます。

📂 タスクのグループ化・複数指定

📁 グループ化: 「グループ」ボタンでグループ行を作成できます。直下のタスクは自動的にグループに属し、インデント（字下げ）して表示されます。

↕️ 折り畳み・展開: テーブル上のグループ名、またはチャート上の空きスペースをダブルクリックして、グループの折り畳み/展開を切り替えられます。

📏 グループサマリー: グループに属するタスクの期間を網羅する「概要バー」が自動的に表示されます。

📅 期間の複数指定: 「期間指定」セルに 04/01-04/05, 04/10-04/15 と入力することで、一つのタスクに複数の期間（断続的な予定など）を設定できます。

🎨 カラー変更: 「色」セルをダブルクリックして、タスクやバーのカラーを簡単に変更できます。

🔍 ズームとカレンダー表示

ズーム・現在ガイド

🔎 メイン画面上部の「1画面の表示枠」と「枠に収める数」を変更することで、画面幅に合わせて自動でズームされます（ウィンドウサイズの変更にも追従）。

🔴 チャート上には「現在時刻」を示す赤い点線が表示されます。

🗓️ 表示は時刻を省いた「日付のみ」に統一され、スッキリとした見た目です。

カレンダー機能

📆 年・月はチャート上部にセンスよくまとまり、各列には「日付」と「曜日」のみが表示されます。

🔵🔴 土曜日（薄い青）、日曜日・祝日（薄い赤）が自動的に強調表示されます。日付をクリックして背景色を消し、「営業日」扱いに変更することも可能です。

🌴 カレンダー上の日付を左クリックすることで、任意の休日・祝日を追加登録できます。

💡 祝日等の日付やヘッダー部分にマウスカーソルを合わせると、名称がツールチップで表示されます。

🧲 横スクロール時は「日単位」で自動スナップし、正確な日付基準でチャートを確認できます。

💻 動作環境・インストール

動作環境: Windows 11 / Windows 10

インストール・実行方法:

配布された MiyaGantt-v2.0.0.exe を任意のフォルダに配置します。

MiyaGantt-v2.0.0.exe をダブルクリックして起動してください。

[!NOTE]
🛡️ 初回起動時の注意（Windows Defenderの警告が出た場合）
Windowsのセキュリティ機能により、青い警告画面「Windows によって PC が保護されました」が表示される場合があります。以下の手順で起動してください。

警告画面内の 「詳細情報」 をクリックします。

右下に現れる 「実行」 ボタンをクリックすると、アプリが起動します。

📖 基本的な使い方

🆕 新規作成: 左上の「追加」ボタン、チャート上のドラッグ操作、または右クリックメニューから新規タスクを作成できます。

⚙️ モード切替: タスクには以下の3種類があり、コンボボックスやテーブルから切り替えられます。

👤 人員モード (手動): 基準となるリソース。

⚡ 案件モード (自動): 人員モードのリソースを自動消化してバーを描画。

📝 メモモード: 集計に影響しない表示専用バー。

🗑️ 削除: バーを右クリックして削除するか、テーブルで選択して「削除」ボタンを押します。

🔧 表示設定: 「表示設定」ボタンから、カレンダーの開始日と全体の表示期間を設定できます。設定はファイルごとに保存されます。

📊 集計表示: 「📊 集計」ボタンから、グループごとの合計日数を「週間・月間・年間」単位で集計したレポートを確認できます。(※案件・メモモード、色が異なるバーは対象外)

🚀 カレンダー移動: 「≪年」〜「年≫」ボタンを使って、素早くジャンプ移動が可能です。

💾 設定保存: 「💾 設定保存」ボタンで、表示設定や最後に使用したファイルの場所を記憶させることができます。

🖨️ 印刷: 「🖨 印刷」ボタンから、指定した期間や行を選択してプレビュー・印刷が可能です。

📁 保存 / 読込: プロジェクト全体を .json 形式で保存・読み込みできます。
