MiyaGantt-v1.0.0

「がんすけ」等と同等の直感的な操作が可能な、高機能ガントチャート作成ツールです。
マウスで直接チャートを操作し、プロジェクトのスケジュールを素早く計画・管理できます。

主な機能

移動: バーを掴んでドラッグすると、日付を変更したり、上下の別のタスク（行）へ自由に移動させたりすることができます。複数のバーを選択している場合は、まとめて移動させることができます。移動後も選択状態は維持されるため、続けて移動が可能です。

範囲選択: チャートの何も無い場所を ドラッグ すると、範囲内のバーをまとめて選択できます（Ctrlキーを押しながらクリックで個別選択も可能）。

マウス操作によるタスク作成: チャートの空きスペースを Shiftキーを押しながらドラッグ すると、その期間に新しいタスクを追加できます。

期間の変更: バーの右端または左端をドラッグすると、期間を自在に伸縮できます。

バーのテキスト編集: チャート上のバーをダブルクリックすると、バーごとに個別のテキスト（名前）を入力・表示できます。

インライン編集: 左側のテーブルの各セル（項目）をダブルクリックすると、そのままテキストを直接入力してタスクを編集できます。

期間の追加（ショートカット）: タスクがある行の何も無い場所をダブルクリックすると、その位置に1日分の期間を素早く追加できます。

タスクのグループ化・複数指定・進捗率

グループ化: 「グループ」ボタンでグループ行を作成できます。グループの下にあるタスクは自動的にそのグループに属し、インデント（字下げ）して表示されます。

折り畳み・展開: テーブル上のグループ名、またはチャート上の空きスペースをダブルクリックすることで、そのグループを折り畳んだり展開したりできます。

グループサマリー: チャート上には、グループに属するタスクの期間を網羅する「概要バー」が自動的に表示されます。

期間の複数指定: テーブルの「期間指定」セルに 04/01-04/05, 04/10-04/15 のように入力することで、一つのタスクに複数の期間を設定できます。

進捗率とカラー: 「進捗(%)」には数値を入力し、「色」セルをダブルクリックすることでカラーを変更できます。バーの内部に進捗率が視覚的に表示されます。

ズーム（1画面の表示枠）機能と「Now」ガイド

メイン画面上部の「1画面の表示枠」と「枠に収める数」を変更することで、画面幅に指定した期間（例：3ヶ月分など）がピッタリ収まるよう自動でズームされます。ウィンドウサイズを変更してもズームは追従します。

チャート上には「現在時刻」を示す赤い点線が表示されます。

表示は時刻を省いた「日付のみ」に統一され、スッキリとした見た目になっています。

視認性の高いカレンダー表示

年・月はチャート上部にセンスよくまとめて表示され、各列には「日付」と「曜日」のみが表示されます。

土曜日（薄い青）と日曜日（薄い赤）、さらに日本の祝日（薄い赤）が自動的に強調表示されます。これらの日付をクリックすることで、個別に背景色を消して「営業日」扱いに変更することも可能です。

カレンダー上の日付を左クリックすることで、任意の休日・祝日を追加登録することができ、これらも同様に強調表示されます。

祝日等の日付やヘッダー部分にマウスカーソルを合わせると、名称がツールチップで表示されます。

横スクロールでカレンダーを移動する際、常に「日単位」の区切りで止まるように自動スナップします。これにより、表示位置が中途半端にならず、常に正確な日付基準でチャートを確認できます。

動作環境

Windows 11 / Windows 10

インストール・実行方法

配布された MiyaGantt-v1.0.0.exe を任意のフォルダに配置します。

MiyaGantt-v1.0.0.exe をダブルクリックして起動してください。

※初回起動時の注意（Windows Defenderの警告が出た場合）
Windowsのセキュリティ機能（SmartScreen）により、青い警告画面「Windows によって PC が保護されました」が表示される場合があります。
これは新規に作成されたアプリでよく発生する画面です。以下の手順で起動してください。

警告画面のテキスト内にある 「詳細情報」 をクリックします。

右下に現れる 「実行」 ボタンをクリックすると、アプリが起動します。

基本的な使い方

新規作成: 左上の「追加」ボタンを押すと即座に空のタスクが追加されます。またチャート上の何もない場所をドラッグ操作するか、右クリックメニューから「タスクの追加」や「期間の追加」を選択することでも作成できます。

編集: 左側のテーブルの各項目（タスク名、進捗、期間指定、色）をダブルクリックし、直接値を入力または設定します。既存のタスクと同じ行の空白部分を右クリックして「期間を追加」することで、1つのタスクに複数の期間（断続的な予定など）を簡単に設定できます。

削除: チャート上のバーを右クリックコンテキストメニューから削除するか、左側の表で選択して「削除」ボタンを押します。

表示設定: 上部の「表示設定」ボタンから、カレンダーの開始日と全体の表示期間（週・月・年）、およびその数を設定できます（チャート全体としての範囲を指定します）。設定はファイルごとに保存されます。

休日・営業日設定: カレンダー上の日付（数字や曜日）の部分を左クリックするだけで、その日を休日として設定・解除できます。元々の日曜日や祝日の日付をクリックした場合は、その列の背景色を消して「営業日」として扱うことができます。追加・変更された設定はカレンダー上に反映されます。

集計表示: 上部の「📊 集計」ボタンから、グループごとの合計日数を「週間・月間・年間」の単位で集計したレポートを確認できます。プロジェクト全体の進捗管理やリソース配分の確認に役立ちます。※「案件モード」のタスクは集計対象外となります。また、バーの色を個別に変更し、その色がタスクの「色」列と異なる場合も集計対象外となります。

カレンダー移動: 「≪年」「＜月」「＜週」「＜日」「📅 今日」「日＞」「週＞」「月＞」「年≫」の各ボタンを使用して、表示位置を素早くジャンプさせることができます。マウスホイールやスライダーでの滑らかな移動に加え、決まった単位での正確な移動が可能です。

設定保存: 「💾 設定保存」ボタンを押すと、表示設定や最後に使用したファイルの保存場所などがアプリ本体に記憶され、次回起動時に自動的に復元されます。

ヘルプ: 「❓ ヘルプ」ボタンを押すと、この説明書（README.md）をアプリ内でいつでも確認できます。

保存: 「保存」ボタンで現在の状態をファイル（.json形式）に保存します。最後に保存した場所が次回のデフォルトになります。

読込: 「読込」ボタンで以前保存したプロジェクトを開きます。最後に読み込んだ場所が次回のデフォルトになります。
